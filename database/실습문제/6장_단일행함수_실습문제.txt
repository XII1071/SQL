1번 문제

1-1. EMP 테이블에서 사원이름(ENAME)이 
     다섯 글자 이상 여섯 글자 미만인 사원 정보(ENAME, EMPNO)를 출력합니다.
1-2. MASKING_EMPNO 열에는 사원번호 앞두자리 외 뒷자리 * 기호로 출력합니다.
     - 앞 두자리 뽑아내기 ( SUBSTR )
     - 뒷 자리 * 로 채워넣기 ( RPAD )
1-3. 그리고 MASKING_ENAME 열에는 사원의 이름 첫 글자만 보여주고 
     나머지 글자 수만큼 * 기호로 출력하세요.
     - 사원 이름의 첫 글자만 뽑아내기 ( SUBSTR ) 
     - 나머지 글자 자리만큼 * 기호로 출력 ( RPAD )

2번 문제

2-1. EMP 테이블에서 사원들의 월 평균 근무일은 21.5일 입니다. 급여 컬럼은 SAL 입니다.
     하루 근무 시간을 8시간으로 보았을때 사원들의 하루 급여(DAY_PAY)와 시급(TIME_PAY)을 
     계산하여 출력하시오. 출력은 사원번호, 사원명, 급여, DAY_PAY, TIME_PAY로 출력하세요.
     DAY_PAY = SAL / 21.5
     TIME_PAY = SAL / 21.5 / 8
2-2. 단, 하루 급여는 소수점 세 번째 자리에서 버리고, 시급은 두 번째 소수점에서 반올림 하세요.
     소수점 N번째 자리에서 버리고 = TRUNC 함수
     소수점 N번째 자리에서 반올림 = ROUND 함수

3번 문제

3-1. EMP테이블에서 사원들은 입사일(HIREDATE)을 기준으로 3개월이 지난 후 첫 월요일에 정직원이 됩니다.
     - 3개월이 지난 후 = ADD_MONTHS()
     - 첫 월요일에 정직원이 된다. = NEXT_DAY()
3-2. 사원들이 정직원이 되는 날짜(R_JOB)를 YYYY-MM-DD 형식으로 
     사원번호, 사원이름, 고용일, 정직원이되는날짜(R_JOB), 추가수당(COMM)으로 출력하시오.
     - TO_CHAR() -> DATE FORMAT 에 맞춰서 출력.
3-3. 단, 추가 수당(COMM)이 없는 사원은 추가 수당을 N/A로 출력하세요.
     - NVL() -> NULL 이면? N/A 출력하라
     

4번 문제 ( CASE-WHEN-ELSE )

4-1. EMP 테이블의 모든 사원을 대상으로 직속 상관의 사원 번호(MGR)를 
     다음과 같은 조건을 기준으로 변환해서 CHG_MGR열에 출력하세요.
     사원번호, 사원이름, 직속상관의사원번호(MGR), CHG_MGR을 출력.
4-2. 직속 상관의 사원 번호가 없을 경우 : 0000
4-3. 직속 상관의 사원 번호 앞 두자리 75일 경우 : 5555
4-4. 직속 상관의 사원 번호 앞 두자리가 76일 경우 : 6666
4-5. 직속 상관의 사원 번호 앞 두자리가 77일 경우 : 7777
4-6. 직속 상관의 사원 번호 앞 두자리가 78일 경우 : 8888
4-7. 그 외 직속 상관 사원 번호의 경우 : 본래 직속 상관의 사원 번호 그대로 출력
